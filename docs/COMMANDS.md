# Commands

## Workflow
- `wsetup` - show version, locations, dependencies, and detected project root
- `wupdate` - pull the latest release from GitHub and reinstall
- `ctfinit` - create a challenge workspace
- `boxprep` - convenience wrapper for `ctfinit --type box`
- `tmuxbox` - create or attach to a tmux workspace
- `wnote` - append timestamped notes
- `wgrep` - grep across the current project or your workspace
- `lootindex` - manage a simple TSV loot index

## Recon and triage
- `scanwrap` - opinionated nmap runner
- `portpeek` - summarize nmap normal output
- `webprobe` - quick HTTP fingerprinting
- `dnspeek` - show DNS records
- `httphead` - fetch headers with redirect following
- `myip` - show local and public IP information
- `netcheck` - check gateway, DNS, internet, and HTTPS
- `flagscan` - look for common flag formats
- `extractall` - extract archives to sane directories
- `hashid` - guess common hash families
- `jwtpeek` - decode JWT header and payload

## HTB helper
- `htbsave` - download and unpack HTB challenge archives with progress, checksum, and default password handling
